# MAT100 - Forelesning: Delbrøksoppspaltning og Integrasjon av Rasjonale Funksjoner

## Kort sammendrag
Forelesningen fokuserer på teknikker for integrasjon av rasjonale funksjoner gjennom delbrøksoppspaltning. Metoden innebærer å dele komplekse brøker ned i enklere delbrøker som kan integreres separat. Foreleseren gjennomgår trinnvis prosessen med polynomdivisjon, faktorisering og systematisk oppløsning av delbrøker, med eksempler som viser hvordan man håndterer ulike typer rasjonale funksjoner.

## Nøkkelbegreper
- Delbrøksoppspaltning: Metode for å dele komplekse rasjonale funksjoner i enklere delbrøker
- Polynomdivisjon: Prosess for å forenkle brøker med høyere grad i teller eller nevner
- Single faktorer: Linjære faktorer uten repetisjon
- Multiple faktorer: Linjære faktorer med repetisjon

## Viktige påstander/teoremer/formler

### Delbrøksoppspaltningsprosess
1. Polynomdivisjon (hvis nødvendig)
2. Faktorisering av nevner
3. Oppdeling i delbrøker basert på faktorenes type

### Delbrøksformer
- Single linjære faktorer: $\frac{A_1}{x-a_1} + \frac{A_2}{x-a_2}$
- Multiple linjære faktorer: $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \frac{A_3}{(x-a)^3}$
- Kvadratiske faktorer: $\frac{Bx+C}{x^2+px+q}$

## Eksempler og typiske feil

### Eksempel 1: Integrasjon med single faktorer
**Integrand:** $\frac{x+2}{x^2 - 5x + 6}$
- Faktoriser nevner: $(x-2)(x-3)$
- Delbrøksoppspaltning: $\frac{A}{x-2} + \frac{B}{x-3}$

### Eksempel 2: Integrasjon med multiple faktorer
**Integrand:** $\frac{x^4 + 3x^3 + 1}{x^2(x^2 + 2x + 2)^2}$
- Krever mer kompleks delbrøksoppspaltning
- Systematisk bestemmelse av koeffisienter

### Typiske feil
- Utelatelse av polynomdivisjon
- Feil faktorisering
- Ukorrekt bestemmelse av delbrøkskoeffisienter

## Forutsetninger og antakelser
- Nevneren kan faktoriseres i linjære/kvadratiske faktorer
- Graden til teller er lavere enn graden til nevner
- Eksistens av reelle røtter

## Metoder/oppskrifter

### Trinn for integrasjon av rasjonale funksjoner
1. Utfør polynomdivisjon om nødvendig
2. Faktoriser nevneren
3. Utfør delbrøksoppspaltning
4. Integrer hver delbrøk separat
   - Single faktorer: $\int \frac{A}{x-a} dx = A \ln|x-a| + C$
   - Multiple faktorer: Mer kompleks integrasjon
   - Kvadratiske faktorer: Krever substitusjon
